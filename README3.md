3. Apa nilai default untuk elemen pada array dengan tipe data int?
jawab: Nilai default untuk elemen pada array dengan tipe data int adalah (0).jika Anda mendeklarasikan array int tanpa menetapkan nilai awal, setiap elemennya akan secara otomatis diisi dengan nilai (0)

5. Array myNumbers memiliki length berbeda untuk setiap barisnya. Apakah panjang array dapat dimodifikasi setelah diinstansiasi?
Jawab: Tidak, panjang array di Java tidak dapat diubah setelah array tersebut di-instansiasi karena array memiliki ukuran yang statis (bersifat tetap)
